To jest kod skryptu "server3.pl"


          #!/usr/bin/perl
	  use HTTP::Daemon;
	  use HTTP::Status;  


	  my $d = HTTP::Daemon->new(
		   LocalAddr => 'localhost',
		   LocalPort => 5004,
	       )|| die;
	  
	  print "Please contact me at: url, ">\n";


	  while (my $c = $d->accept) {
	      while (my $r = $c->get_request) {
		  if ($r->method eq 'GET') {
		      
		      $file_s= "pages/helloworld.html";    
		      $c->send_file_response($file_s);

		  }
		  else {
		      $c->send_error(RC_FORBIDDEN)
		  }

	      }
	      $c->close;
	      undef($c);
	  }

Jeśli to czytasz, to znaczy, że nawiązałeś połączenie za pomocą przeglądarki internetowej

Skrypt ten jest prostym serwerem HTTP, odpowiadającym na żądania GET i zwracający wtedy plik pages/helloworld.html, w przeciwnym razie zwraca błąd 403 Forbidden
